# assembly-snake

assembly-snake 包含三部分：

* snake.cpp
  最初的成果，使用 C 嵌套汇编编写
* snake.asm
  第二个版本，纯汇编编写
* super-snake.asm
  第三个版本，纯汇编编写，支持单人游戏和双人游戏
